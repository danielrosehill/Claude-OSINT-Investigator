# Entity Profiles

Detailed profiles of people, organizations, accounts, and other entities of interest.

## For Claude

Create comprehensive profiles using this template:

```markdown
# Entity Profile: [Name/Handle]

## Basic Information
- **Type**: Person / Organization / Account / Other
- **Known Names/Aliases**:
- **First Identified**:
- **Relevance to Investigation**:

## Identifiers
- Social media handles
- Email addresses
- Phone numbers
- Usernames
- Other identifiers

## Summary
[Brief overview of who/what this entity is]

## Detailed Findings
[Organized findings with evidence citations]

## Connections
[Links to other entities - reference network-maps/]

## Timeline
[Key events - reference timeline/]

## Open Questions
[What we still don't know]

## Sources
[List of evidence files supporting this profile]
```

Update profiles as new information emerges. Maintain change log for significant updates.
