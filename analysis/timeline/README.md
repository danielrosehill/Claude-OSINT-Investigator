# Timeline Analysis

Chronological reconstruction of events.

## For Claude

Create and maintain timelines in Markdown format:
- Use consistent date format: `YYYY-MM-DD HH:MM (timezone)`
- Link each event to supporting evidence
- Mark uncertain dates with `~` prefix
- Include source citations

Example format:
```markdown
## Timeline: [Subject]

| Date | Event | Source | Confidence |
|------|-------|--------|------------|
| 2024-01-15 | Account created | evidence/screenshots/... | High |
| ~2024-02 | First public post | evidence/raw/... | Medium |
```
