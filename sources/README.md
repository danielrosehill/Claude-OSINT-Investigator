# Sources

Index of information sources used in the investigation.

## Structure

- **verified/** - Sources confirmed as authentic/reliable
- **unverified/** - Sources requiring validation
- **archived/** - Archived copies of online sources (for preservation)

## For Claude

Maintain a source index tracking:
- Source name/description
- URL or location
- Date accessed
- Reliability assessment
- What information was obtained

Create `sources-index.md` to maintain a master list of all sources consulted.
